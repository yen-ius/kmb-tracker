📋 설정 방법
  1. Google Drive에서 새 스프레드시트 생성 (원하는 폴더 어디든 OK)
  2. 상단 메뉴 → 확장 프로그램 → Apps Script 클릭
  3. 함께 받은 KMB_Apps_Script.gs 코드 전체를 붙여넣기 후 저장
  4. 배포 → 새 배포 → 유형: 웹 앱 선택
  5. 실행 계정: 나 / 액세스: 모든 사용자 선택 후 배포
  6. 팝업에서 액세스 승인 → 구글 계정 로그인
  7. 생성된 웹 앱 URL을 아래에 붙여넣기
URL을 입력하고 연결 버튼을 누르세요.
